# Installation guide

```
wget https://github.com/prendssoin/ec2-configurator/archive/v1.0.0.tar.gz && mkdir ec2-configurator && tar -xzf v1.0.0.tar.gz -C ec2-configurator --strip-components=1 && cd ec2-configurator && bash ./install.sh && cd .. && rm -rf *
```
